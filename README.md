## Converter CSV
<br>
This is a simple script that read a file JSON and convert this file to CSV

<br>
<strong>To install dependencies</strong>

```js
npm install 
```

<br>
<strong>To execute script</strong>

```js
npm run dev
```

<strong>Technologies</strong>

- NodeJs
- @json2csv/transform
- @json2csv/plainjs
